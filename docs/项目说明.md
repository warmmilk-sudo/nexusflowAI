# NexusFlow AI - 项目说明

## 🎯 项目概述

**NexusFlow AI** 是一个基于私有知识库的智能邮件助手，旨在通过 AI 自动处理繁杂的医疗/商务邮件，实现从"阅读"到"草拟回复"的自动化，但保留人工确认的控制权。

### 核心价值
- **自动化**：减少 80% 的邮件处理时间
- **智能化**：AI 驱动的意图识别和内容生成
- **知识化**：基于企业知识库的精准回复
- **人性化**：保留人工决策权，AI 作为辅助

---

## ✨ 核心功能

### 🧠 智能收件箱
- **自动分类**：AI 自动识别邮件类型并打标签（销售/技术/支持/垃圾）
- **一句话摘要**：在邮件列表直接显示 50 字核心内容摘要
- **智能回复**：基于知识库自动生成专业回复草稿
- **来源追溯**：明确标注引用的知识库文档
- **置信度评分**：显示 AI 分析的可信度

### 📤 批量外发
- **CSV 导入**：批量上传客户信息
- **超个性化**：基于客户痛点生成定制化邮件
- **活动配置**：支持多种活动类型（产品发布、合作邀请等）
- **批量生成**：一键生成数百封个性化邮件
- **预览审核**：人工审核每封邮件后再发送

### 📚 知识库管理
- **文档上传**：支持 PDF、DOCX、TXT 格式
- **自动分块**：智能切分文档便于检索
- **语义搜索**：基于关键词的相关内容检索
- **实时统计**：监控存储使用率和文档数量

### 📊 数据仪表盘
- **关键指标**：总触达量、回复率、待处理草稿、活跃线索
- **趋势分析**：每周邮件发送量 vs 回复量图表
- **快捷操作**：一键跳转到各功能模块

### ⚙️ 设置中心
- **多语言支持**：中文/英文界面切换
- **API 配置**：管理 Gemini API Key
- **邮箱配置**：设置收发邮件的邮箱账号
- **本地存储**：所有配置仅存储在浏览器本地

## 🏗️ 技术架构

### 前端技术栈
- **React 19** + TypeScript - 现代化前端框架
- **Vite 6** - 快速构建工具
- **Tailwind CSS** - 实用优先的样式框架
- **Lucide React** - 精美的图标库
- **Recharts** - 数据可视化图表

### 后端技术栈
- **Node.js 18+** - 服务器运行时
- **Express** - Web 应用框架
- **Google Gemini API** - AI 大语言模型
- **IMAP/SMTP** - 邮件协议支持
- **RAG 技术** - 检索增强生成

### 核心技术
- **RAG (检索增强生成)**：结合知识库的 AI 生成
- **多语言国际化**：完整的中英文支持
- **实时 AI 分析**：邮件意图识别和内容生成
- **模块化架构**：前后端分离，易于扩展

## 🔄 核心工作流

### 智能邮件处理流程
```
📧 收到邮件
    ↓
🧠 AI 意图识别 (Gemini)
    ↓
🔍 RAG 检索知识库
    ↓
✍️ 生成回复草稿
    ↓
👤 人工审核确认
    ↓
📤 发送回复
```

### 批量外发流程
```
📊 上传客户数据 (CSV)
    ↓
⚙️ 配置活动参数
    ↓
🤖 AI 批量生成个性化邮件
    ↓
👁️ 预览和编辑
    ↓
✅ 批准并发送
```
