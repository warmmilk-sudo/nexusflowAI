# NexusFlow AI - 智能邮件助手

基于私有知识库的智能邮件助手，通过 AI 自动处理繁杂的医疗/商务邮件，实现从"阅读"到"草拟回复"的自动化。

## 🚀 快速开始

```bash
# 安装依赖
npm run install:all

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 开始使用。

## ✨ 核心功能

- 🧠 **智能收件箱** - 自动分类、摘要、智能回复
- 📤 **批量外发** - 个性化邮件批量生成
- 📚 **知识库管理** - 文档上传和语义检索
- 📊 **数据仪表盘** - 关键指标和趋势分析
- ⚙️ **设置中心** - 多语言和邮箱配置
